# ReactJS-App

This is a ReactJS application developed for the SoftUni Regular Exam. It features a culinary platform where users can browse a variety of recipes. After registering, users gain full access to create, edit, and delete their own recipes.

# Features included: 
- [x] Authentication:
    - Register
    - Login
    - Logout
    - Persisted user session (localStorage)
    - Route guards for guest/private routes

- [x] Recipes:
    - Browse all recipes (Catalog)
    - View recipe details
    - Create new recipe (authenticated users)
    - Edit own recipes
    - Delete own recipes
    - List only my recipes
    - Show latest 3 recipes on home page

- [x]Other
    - 404 Page (Not Found)
    - Modular architecture
    - Custom reusable form hook (useForm)
    - Custom localStorage hook (usePersistedState)
